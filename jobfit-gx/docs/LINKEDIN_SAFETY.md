# LinkedIn Safety

JobFit GX is not a LinkedIn bot.

It does not:

- Mass scrape LinkedIn.
- Apply to jobs.
- Message recruiters.
- Bypass LinkedIn restrictions.
- Read pages in the background.

The safe MVP workflow is manual:

1. Open a job page.
2. Click JobFit GX.
3. Paste job text or explicitly extract visible page text.
4. Analyze locally.

Keep usage personal, manual, and respectful of each site's terms.
